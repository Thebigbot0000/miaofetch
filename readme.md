## 🐱 `miaofetch`

📎 Fetch your system information, with a cool cat

</div>


### 📥 Installation

1. Clone Repository

```bash
git clone https://github.com/aThebigbot/miaofetch.git
cd miaofetch
```

2. Run the Installation Process

```bash
sh install
```

### 🥤 How To Use
Now just run this in your terminal.
```bash
miaofetch
```
