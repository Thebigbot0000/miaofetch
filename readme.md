## Miaofetch, Miao!

Fetch your system information, with a nice cat! 

### Installation

1) Clone repository

```bash
git clone https://github.com/Thebigbot0000/miaofetch.git && cd miaofetch
```

2) Start `install`

```bash
sh install
```

### How to use

```bash
# For run it, just write this in your console/terminal:
miaofetch
```
